# FessHub Changelog & Roadmap

##  Version History

### Version 1.0.0 (December 7, 2025) - Initial Release
**New Features:**
-  Core framework architecture
-  Blox Fruits support with basic automation
-  Meme Sea basic structure
-  Modern draggable UI
-  Anti-spam protection
-  Queue teleport (save settings on server hop)
-  Error handling system
-  Multi-game detection
-  Settings customization

**Blox Fruits Features:**
-  Auto Farm (level grinding)
-  Bring Mobs system
-  Fast Attack mode
-  Distance-based farming
-  Anti-AFK system
-  Enemy detection & sorting
-  Teleport & Tween movement
-  Weapon equip system

**Known Issues:**
-  BringMobs may trigger anti-cheat on some executors
-  UI needs more features and tabs
-  Auto Quest not implemented yet

##  Bug Reports

**How to Report:**
1. Join Discord: discord.gg/fesshub
2. Go to #bug-reports channel
3. Use the template:
   `
   Game: [Blox Fruits/Meme Sea]
   Executor: [Your executor name]
   Issue: [Describe the bug]
   Steps: [How to reproduce]
   Screenshot: [Optional]
   `

**Current Known Bugs:**
- BringMobs detection on some executors
- UI dragging issues on mobile
- Rare teleport failures
